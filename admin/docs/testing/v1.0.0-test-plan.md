## 1. 测试范围

### 1.1 测试对象
- 后端 API（Express + SQLite）
- 前端页面（React 组件）
- E2E 用户流程（Playwright）

### 1.2 测试类型
| 类型 | 工具 | 位置 |
|------|------|------|
| 后端接口测试 | Vitest + Supertest | admin/backend/tests/ |
| 前端组件单元测试 | Vitest + Testing Library | admin/frontend/src/test/ |
| E2E 端对端测试 | Playwright | admin/frontend/tests/e2e/ |
